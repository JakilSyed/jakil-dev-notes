

1. Initialize a Git repository
git init

2. git status

3.Stage all files for commit
git add .

4. Commit the staged files
git commit -m "Initial commit: springboot-rabbitmq-producer-consumer-demo"

5. Rename the default branch from 'master' to 'main'
git branch -M main

6. Add the remote GitHub repository (replace with your actual repo URL)
git remote add origin https://github.com/YourUsername/YourRepoName.git
git remote add origin https://github.com/JakilSyed/jakil-dev-notes.git

7. Push your local main branch to GitHub and set upstream tracking
git push -u origin main
